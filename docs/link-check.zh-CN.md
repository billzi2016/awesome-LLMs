# 链接检查

新增或更新资源前，使用本清单检查。

## 必查项

- 链接可以正常打开。
- 链接指向目标资源。
- 标题、作者、会议/期刊、年份、仓库身份或 skill 名称与页面一致。
- 代码或仓库链接尽量指向作者、机构或平台官方页面。
- 数据集、模型、文档和 registry 链接应指向官方页面，而不是镜像站或转贴页。
- 资源不需要把受保护内容复制到本仓库。

## 优先来源

1. arXiv、DOI、出版社页面、官方论文页或平台 / 提供方官方文档
2. 官方 GitHub、机构或平台仓库
3. 官方项目、registry、marketplace 或 catalog 页面
4. 官方文档
5. 官方数据集、benchmark 或打包 / runtime 页面

## 风险信号

- 只有下载入口且所有权不清楚的页面
- 非官方镜像
- 没有许可证说明的模型权重
- 转贴的文档页面
- 复制的 PDF 或课程材料
- 间接分享的私有 marketplace 内容
- 异常跳转
- 页面内容和资源标题不匹配

不确定时，不要收录，先核验来源。
