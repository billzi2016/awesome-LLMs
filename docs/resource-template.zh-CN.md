# 资源模板

## 论文

```md
- **论文标题** - 作者，会议/期刊 年份。一句话客观说明方法、任务或贡献。[[论文](link)] [[代码](link)]
```

## 仓库或代码库

```md
- [仓库名称](link) - 与仓库主题相关的工具或实现，说明范围和主要使用场景。
```

## 教程或指南

```md
- [教程标题](link) - 覆盖某个概念、实现细节或应用方向的学习资源。
```

## 数据集、Benchmark 或评估资源

```md
- [数据集或 Benchmark 名称](link) - 面向某个任务的评估资源，可说明领域、模态和访问方式。
```

## 官方文档、规范或平台体系

```md
- [平台或文档名称](link) - 与仓库主题直接相关的官方文档、规范或平台原生体系。
```

## Registry、Marketplace 或 Catalog

```md
- [目录名称](link) - 与仓库主题直接相关的发现、分发或检索入口。
```

## 模板、脚手架或工作流示例

```md
- [资源名称](link) - 可复用的模板、脚手架、工作流或示例；如已知，可说明目标 runtime 或主要场景。
```

## 应用

```md
- **应用论文或项目** - 作者，会议/期刊 年份。核心主题在具体领域或任务中的应用。[[论文](link)] [[代码](link)]
```

## 描述规则

- 描述必须客观。
- 说明资源为什么属于本列表。
- 当上下文重要时，补充 runtime 边界、打包价值或工作流角色。
- 避免宣传语言。
- 优先使用原始来源。
- 不收录未核验链接。
