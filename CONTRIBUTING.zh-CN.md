# 贡献指南

感谢你帮助改进本项目。

## 可以提交什么

适合提交的资源包括：

- 与本项目主题范围直接相关的资源；
- 官方仓库、文档、规范或标准；
- 持续维护的库、工具、模板或脚手架；
- 教程、课程笔记、技术指南或打包指南；
- 有明确相关性的 benchmark、数据集、评估协议、registry 或 marketplace；
- 明确属于本仓库主题范围的代表性应用、工作流或可复用 skill 示例。

请不要提交：

- 与项目范围无关的资源；
- 只有弱相关性的泛背景材料；
- 与本仓库主题没有清晰关系的泛 AI 工具目录；
- 没有技术细节的宣传页面；
- 复制的受保护内容、镜像数据集、非官方模型权重或私有 marketplace 内容；
- 无法核验的链接。

## 条目格式

论文条目：

```md
- **论文标题** - 作者，会议/期刊 年份。一句话客观说明方法、任务或贡献。[[论文](link)] [[代码](link)]
```

仓库、文档或实现条目：

```md
- [仓库名称](link) - 一句话客观说明范围、实现和使用场景。
```

工作流、skill 或示例条目：

```md
- [资源名称](link) - 一句话客观说明目标 runtime、工作流边界和主要用途。
```

## PR 检查清单

- [ ] 资源与项目主题范围直接相关。
- [ ] 链接真实且可以访问。
- [ ] 标题、作者、年份和相关链接在适用时正确。
- [ ] 描述客观、简洁。
- [ ] 不是重复条目。
- [ ] 条目格式符合现有规范。
- [ ] 如有必要，英文和中文 README 已同步更新。
- [ ] 资源不会引入版权、许可证、隐私或平台规则风险。

## 链接与版权要求

优先使用原始来源：

- arXiv、DOI、出版社页面或官方论文页；
- 作者或机构官方代码仓库；
- 数据集官方页面；
- 官方文档或平台 / 提供方页面；
- benchmark、registry、marketplace 或项目官方页面。

不要把受版权保护的 PDF、书籍、课程材料、数据集、模型权重、图片或受保护的 marketplace 内容复制到本仓库。

## 双语同步

本项目使用：

- `README.md` 作为英文版本；
- `README.zh-CN.md` 作为简体中文版本。

新增或删除资源时，请保持两个文件内容同步，除非该修改只影响某个语言版本的说明文字。
